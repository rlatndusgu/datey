# 💕 Datey — 데이티

커플을 위한 기념일 & 데이트 코스 플래닝 웹 서비스

---

## 프로젝트 소개

Datey는 커플이 기념일을 놓치지 않고, 데이트 코스도 쉽게 계획할 수 있도록 도와주는 웹 서비스입니다.  
처음 만난 날을 입력하면 100일, 200일, 1주년 등 주요 기념일이 자동으로 계산되어 달력에 표시되고,  
지역·예산·분위기·시간대를 선택하면 맞춤 데이트 코스를 추천받을 수 있습니다.

---

## 페이지 구성 (총 6페이지)

| 파일 | 페이지 이름 | 주요 기능 |
|------|------------|----------|
| `index.html` | 홈 | 처음 만난 날 입력, D+N 카운터 표시 |
| `calendar.html` | 캘린더 | 기념일 및 저장된 데이트 표시, 날짜 클릭 |
| `anniversary.html` | 기념일 | 100일·200일·1주년 등 D-day 목록 |
| `filter.html` | 필터 | 지역·예산·분위기·시간대 선택 |
| `result.html` | 추천 결과 | 필터 적용된 데이트 코스 카드 |
| `detail.html` | 코스 상세 | 장소 순서·팁·예산 확인, 캘린더 저장 |

---

## 주요 기능

### 1. 기념일 자동 계산
- 처음 만난 날을 기준으로 100일, 200일, 300일 등 100일 단위 기념일 계산
- 1주년 ~ 10주년 자동 계산
- 달력에 배지로 표시, 기념일 페이지에서 D-day 확인 가능

### 2. 데이트 코스 추천 (필터 기반)
- 조건: 지역 (홍대 / 강남 / 인사동 / 이태원 / 신촌)
- 조건: 예산 (3만원 이하 / 3~7만원 / 7만원 이상)
- 조건: 분위기 (로맨틱 / 캐주얼 / 액티브 / 문화적)
- 조건: 시간대 (오전 / 오후 / 저녁 / 밤)
- 15개 코스 데이터, 조건에 따라 동적 필터링

### 3. 동적 캘린더
- JavaScript DOM으로 달력 생성
- 이전/다음 달 이동 가능
- 기념일 날짜와 저장된 데이트 코스를 색상으로 구분

### 4. localStorage 저장
- 처음 만난 날 저장
- 선택한 데이트 코스를 날짜별로 저장
- 페이지 새로고침 후에도 데이터 유지

### 5. 반응형 레이아웃
- CSS Grid: 달력, 코스 카드 그리드
- CSS Flexbox: 네비게이션, 필터 버튼, 기념일 목록
- 모바일 / 데스크탑 모두 지원

---

## 사용 기술

- HTML5
- CSS3 (Grid, Flexbox, 반응형 미디어쿼리)
- JavaScript (DOM 조작, 이벤트 처리, localStorage)
- Google Fonts (Nanum Gothic)

별도 서버나 프레임워크 없이 순수 HTML/CSS/JS만 사용합니다.

---

## 실행 방법

1. 이 폴더를 다운로드합니다.
2. `index.html`을 브라우저에서 엽니다.
3. 처음 만난 날을 입력하고 시작하면 됩니다.

> 서버 설치나 빌드 과정 없이 바로 실행 가능합니다.

## 데이트 코스 데이터 (`data.js`)

총 15개의 코스가 포함되어 있으며 각 코스는 다음 정보를 가집니다.

```javascript
{
    id: 1,
    title: "홍대 카페 & 공방 데이트",
    region: "홍대",          // 지역
    budget: "low",           // 예산 (low / mid / high)
    mood: "casual",          // 분위기 (romantic / casual / active / cultural)
    time: "afternoon",       // 시간대 (morning / afternoon / evening / night)
    emoji: "☕",
    summary: "...",
    description: "...",
    places: [...],           // 장소 목록
    budget_detail: "...",    // 예상 비용 상세
    tip: "..."               // 팁
}
```
